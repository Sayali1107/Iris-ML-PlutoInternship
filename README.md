# Iris Flower Classification - ML Project
Machine Learning project comparing 3 models on the Iris dataset.
Part of Pluto Academy AI/ML Internship Program.

## Dataset
Source: https://www.kaggle.com/datasets/uciml/iris

## Models Used
- Logistic Regression
- Random Forest
- KNN (K Nearest Neighbors)

## Tools Used
Python, Scikit-learn, Pandas, Matplotlib, Seaborn, Google Colab

## Results
Random Forest performed the best and had the highest accuracy among all 3 models.
